A frequently asked question with integrations: how up to date is my data, really? The answer is yours to decide: per object, tailored to your process.

## Real-time synchronisation

With real-time sync, data is updated immediately as soon as something changes in Salesforce. If an account manager sets a deal to *Closed-Won*, the invoice or order is created instantly in Exact Online. Ideal for immediate invoicing, critical stock information and customer service.

## Scheduled synchronisation

With scheduled sync, data is processed at fixed intervals, for example every hour or daily. This is useful for batch-processing large data volumes, for a manual check before data goes to the bookkeeping, and for a predictable system load.

## Mix both as needed

You do not have to choose: you set the strategy per object type. Customer data can synchronise in real time, for example, while orders are processed once a day. The configuration is done through the visual mapping interface.

## Error reporting built in

Built-in logging provides immediate error reporting, so any problems are easy to resolve.
